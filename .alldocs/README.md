# .alldocs Directory

Welcome to `.alldocs`, the raw documentation repository for the **Handa** (AURA) Stroke Rehabilitation App. 

## Purpose
While the `.knowledge` folder acts as the condensed, highly actionable "wiki" for AI coding assistants, `.alldocs` serves as the **source of truth archive**. It contains the original Product Requirements Documents (PRDs), raw LLM chat histories where the architecture was planned, reference implementation code, and the theoretical frameworks for the digital brain clone.

## Folder Structure

*   **/Agent_Subsystems/**: Contains individual markdown specifications for the specialized modular components of the AI background agent (e.g., extracting emotions, updating the relationship tree).
*   **/Chat_Histories/**: Unedited transcripts of previous LLM sessions (Kimi, Gemini, and OpenCode) detailing the design and scaffolding processes.
*   **/Core_Architecture/**: The foundational product documents, including the PRD, the Master Prompt, and the Psychological/Memory Architecture definitions.
*   **/Reference_Code/**: Zip archives and extracted references of prior implementations (e.g., the original React/Node WebSocket prototype).
*   **/Reference_Materials/**: Generic AI skills and prompts used for contextual framework operations.
