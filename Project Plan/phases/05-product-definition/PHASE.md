# Phase 6/15 — Product Definition

**Status:** COMPLETE ✅

## Vision (1 sentence)
A digital therapist that knows your father better than any human could — because it remembers everything, patterns everything, and adapts everything.

## Mission (1 sentence)
Build an AI-powered speech therapy companion with a living digital brain that tracks every dimension of patient progress across biological, relational, and emotional domains.

## Target Audience
- **Primary:** Father (Sinhala-speaking male, ~60s, post-stroke aphasia + apraxia)
- **Secondary (Phase 3):** Other stroke survivors with aphasia
- **Tertiary (Phase 3):** Speech therapists monitoring patient progress

## User Personas

### Persona 1: The Patient (Father)
- **Name:** Father
- **Age:** ~60s
- **Background:** Post-stroke, fluent speaker before. Now has apraxia + aphasia.
- **Goals:** Speak better, regain independence, reduce shame
- **Pain points:** Frustration with word-finding, tires easily, complex family emotions
- **AI strategy:** Frame as "work" not therapy, start with easy wins, avoid shame triggers (money, visitors)

### Persona 2: The Builder (You/Son)
- **Role:** Developer + caregiver
- **Goals:** Build a tool that helps father speak, automate memory so AI improves every session
- **Pain points:** Time-consuming manual memory updates, complex family dynamics affecting therapy
- **AI strategy:** Background agent handles memory; you focus on features

### Persona 3: The Therapist (Future)
- **Role:** Speech therapist (Phase 3+)
- **Goals:** Monitor patient progress, adjust difficulty, review reports
- **Pain points:** No visibility into daily performance between sessions
- **AI strategy:** Caregiver report auto-generated with wins + areas of work

## KPIs (7)

| # | KPI | Target | Measurement |
|---|-----|--------|-------------|
| 1 | Word accuracy improvement | +20% over 4 weeks | Word mastery tracker |
| 2 | Word onset latency reduction | -30% over 4 weeks | Speech metrics |
| 3 | Session completion rate | >80% sessions complete full 20 min | Session logs |
| 4 | Frustration event reduction | -40% over 4 weeks | Amygdala trigger tracking |
| 5 | Background agent update time | <5 min post-session | Agent completion timestamp |
| 6 | Memory injection accuracy | AI correctly references >2 previous sessions | Prompt evaluation |
| 7 | Patient willingness to start | >90% sessions started without resistance | Session logs |

## Success Metrics
- AI response latency < 1 second
- Struggle detection accurately triggers cueing ladder
- Background agent updates brain within 5 min of session end
- Relationship weights adjust ±0.05 per session
- Caregiver report accurately reflects wins + frustration events
