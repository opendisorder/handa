# GLOSSARY — AURA Speech Therapy (Handa)

| Term | Definition |
|------|-----------|
| **Handa** | Project codename; Sinhala for "ready/prepared" |
| **AURA** | Application name; AI-driven Utterance Rehabilitation Assistant |
| **Aphasia** | Language disorder affecting speech production/comprehension |
| **Apraxia** | Motor disorder affecting speech muscle coordination |
| **Gemini Live API** | Google's real-time bidirectional audio streaming API |
| **MediaPipe Face Mesh** | On-device facial landmark detection for struggle analysis |
| **Cueing Ladder** | 5-level progressive scaffolding (silent articulation → direct production) |
| **4-7-8 Breathing** | Box breathing protocol: inhale 4s, hold 7s, exhale 8s |
| **Thilina** | AI therapist persona name |
| **Digital Brain** | 10-region biological patient model (prefrontal cortex, amygdala, broca, etc.) |
| **Relationship Tree** | Weighted graph + per-person files tracking patient relationships |
| **Entity Graph** | Topics/concepts with emotional valence (joy, shame, trigger) |
| **Memory Injection** | 200-300 word compressed brain summary prepended to system prompt |
| **Background Agent** | Post-session analysis system that updates brain files autonomously |
| **Brain Region File** | Living .md document for each of 10 brain regions |
| **Struggle Level** | 0-4 scale from MediaPipe analysis (0=relaxed, 4=high frustration) |
